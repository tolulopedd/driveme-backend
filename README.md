# DriveMe API

Express + Prisma backend for the DriveMe MVP.

## Modules

- Auth
- Users
- Drivers
- Driver onboarding
- Bookings
- Trips
- Locations
- Payments
- Notifications
- Admin

## Local commands

```bash
npm run dev -w @driveme/api
npm run prisma:generate -w @driveme/api
npm run prisma:migrate -w @driveme/api
npm run seed -w @driveme/api
```
